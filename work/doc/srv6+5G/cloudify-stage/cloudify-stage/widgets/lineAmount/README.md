## 链路查询

主要展示以下用户参数：

- **网关**：网关唯一编号
- **名称**：网关名称
- **线路**：网关线路名称
- **状态**：线路状态
- **流入pps**：每秒流入分组数据包数量
- **流出pps**：每秒流出分组数据包数量
- **流入bps**：流入速率
- **流出bps**：流出速率
- **更新时间**：线路更新时间

根据需求需要三个按钮：

- **更新**：该按钮绑定名字为bp_line_query.yaml的蓝图文件，创建一个名为lineAmount的部署，无参数输入

  


#### widget参数设置：

- **定时刷新**：30s
- **分页大小**：10
- **排序**：按照网关名称排序
- **搜索**：指定名称查找网关
