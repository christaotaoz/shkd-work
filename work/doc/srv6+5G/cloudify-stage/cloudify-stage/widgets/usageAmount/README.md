## 用户查询

主要展示以下用户参数：

- **账号**
- **登录ip**：最近一次该账号的登录ip地址
- **登录时长**
- **上传流量**
- **下载流量**
- **总流量**
- **更新时间**：账号信息更新时间
- **操作**：提供用户账号的更新和删除操作（需要插件支持）
- **隐藏信息**
  - **创建时间**：在本系统中创建账号的时间
  - **NAS IP**：账号登录的网关ip地址
  - **允许登录时间段**
  - **允许每月在线时间**
  - **允许最大在线时间**
  - **允许最大使用流量**
  - **用户组**：账号所属的用户组

根据需求需要三个按钮：

- **更新**：该按钮绑定名字为Traffic_query.yaml的蓝图文件，创建一个名为usageAmount的部署，默认输入用户为“all”，月份为 “0”

- **用户更新**：绑定名字为update_user.yaml的蓝图文件，创建一个名为update_user的部署，接受输入用户名，密码，用户组，和用户限制。用于更新用户配置，升级账号。

- **用户删除**：绑定名字为delete_user.yaml的蓝图文件，创建一个名为delete_user的部署，接受用户输入用户名，首次创建部署时需要填入用户名，后续使用按钮时，自动填入。

  

#### widget参数设置：

- **定时刷新**：10s
- **分页大小**：10
- **默认排序**：按照用户的创建时间
- **指定排除**：主要展示参数都可指定排序
- **用户搜索**：提供按用户名查找用户
- **操      作** ：指定用户的升级和删除
